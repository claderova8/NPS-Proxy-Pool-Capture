# nps-代理池抓取

> 打点代理池，支持弱口令检测与 SOCKS5 隧道抓取

## 特性

- 🎯 自动抓取高可用 HTTP/HTTPS 代理  
- 🔓 弱口令检测 (HTTP Basic / Digest)  
- 🧩 通过本地 SOCKS5 隧道抓取目标站点  
- 🚀 支持多线程并发

## 安装

```bash
git clone https://github.com/claderova8/nps-proxy-pool.git
cd nps-proxy-pool
pip install requests tqdm
```
